# threejs-basics

A simple Three.js project that only renders a red cube on the Screen.

![image](https://user-images.githubusercontent.com/62032328/228676758-df57b146-a81a-4f72-bfff-3c8de79c4c55.png)


## How to run it

1. clone this repository
2. open the `index.html` file on your browser (you can do it with a double click on the file or dragging it to the browser)
